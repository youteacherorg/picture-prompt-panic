# 📸 Picture Prompt Panic 😱

An interactive IELTS Speaking Part 2 classroom game for EFL teachers.  
Built for smartboard use. Runs completely offline from a USB stick.  
No installation. No internet required. Just open `index.html` in Chrome.

---

## What it does

Each round displays a photograph paired with a matching Part 2 cue card topic.  
A **Language Boost** panel provides a collocation, idiom, and sentence starter  
for each topic — with Chinese translations — to help students attempt Band 6+ language.

The teacher controls all pacing from the board:

- **1-minute prep timer** — student thinks and makes notes
- **2-minute speaking timer** — student responds
- **Live notes panel** — teacher types what the student says in real time for post-response analysis
- **20 matched image/prompt pairs** — auto-shuffles, no repeats per session
- **Light/dark mode**, font size controls, adjustable text size for notes

## Setup

1. Download or clone this repository
2. Place your images in an `images/` folder in the same directory as `index.html`
3. Name images exactly as listed below
4. Open `index.html` in Chrome — works offline, no server needed

## How to run a round

| Key / Button          | Action                  |
| --------------------- | ----------------------- |
| **SPACEBAR**          | Advance through stages  |
| **Generate Round**    | Show image + cue card   |
| **Start 1 Min Prep**  | Begin preparation timer |
| **Start 2 Min Speak** | Begin speaking timer    |
| **Notes**             | Toggle live notes panel |
| **Reset All**         | Start a new session     |
